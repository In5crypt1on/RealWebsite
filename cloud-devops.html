DevOps is a culture and set of practices that combines software development (Dev) and IT operations (Ops). It aims to shorten the development life cycle and provide continuous delivery with high software quality. The languages and tools in this category are used to automate infrastructure, build deployment pipelines, and manage cloud services.

HCL (Terraform)

HashiCorp Configuration Language (HCL) is a declarative language used by Terraform to define and provision cloud infrastructure as code.

  • Declarative: You describe the desired end state of your infrastructure, and Terraform figures out how to create, update, or destroy resources to reach that state.
  • Multi-Cloud: A single configuration can be used to manage infrastructure across multiple cloud providers like AWS, Azure, and Google Cloud.
  • State Management: Terraform keeps track of your managed infrastructure in a state file, which it uses to plan and apply changes.
# Defines an AWS EC2 instance
resource "aws_instance" "web" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = "t2.micro"
}

Go (Golang)

A statically typed, compiled language designed at Google. Its simplicity, performance, and strong support for concurrency make it ideal for DevOps tooling.

  • Concurrency: "Goroutines" and "channels" make it easy to write programs that do many things at once, perfect for network services.
  • Fast Compilation: Compiles extremely quickly to a single static binary, making deployment simple and efficient.
  • DevOps Tooling: Many foundational DevOps tools, including Docker, Kubernetes, and Prometheus, are written in Go.
package main
import "fmt"

func main() {
    fmt.Println("Hello, DevOps!")
}

Python

While also a leader in data science, Python's simplicity and extensive libraries make it a go-to language for automating cloud tasks and writing operational scripts.

  • Scripting & Automation: Its easy-to-read syntax makes it perfect for writing scripts to automate repetitive tasks, manage deployments, or orchestrate services.
  • Cloud SDKs: Libraries like Boto3 (for AWS), Azure SDK for Python, and Google Cloud Client Libraries provide powerful, high-level access to cloud provider APIs.
  • Glue Language: Often used to connect different tools and services together in a DevOps pipeline.
import boto3

# Create an S3 client
s3 = boto3.client('s3')

# List all S3 buckets
response = s3.list_buckets()
for bucket in response['Buckets']:
    print(f'  {bucket["Name"]}')
